Atualize o sistema SaaS existente **ImobCore OS** criando um módulo completo chamado:

💰 FINANCEIRO INTELIGENTE

Este módulo deve incluir:

* Gestão de receitas e despesas
* Previsão orçamentária mensal
* Fluxo de caixa
* Score financeiro automático
* Insights com IA (Síndico Virtual)
* Dashboard visual estilo dark (tipo Nubank)

Stack obrigatória:

* Backend: Node.js + Express
* Banco: Supabase
* Frontend: React (com Tailwind)
* Gráficos: Chart.js
* IA: endpoint já existente /api/sindico/chat

3. LÓGICA PRINCIPAL

Fluxo de caixa:

* somar receitas
* somar despesas
* calcular saldo

Previsão:

* média dos últimos 3 meses
* projetar próximos 3 meses

Score financeiro:

Começa em 100 e reduz:

* inadimplência > 10% → -30
* saldo negativo → -40
* despesas > receitas → -20

Classificação:

> 80 excelente
> 60 bom
> 40 atenção
> <40 crítico

────────────────────────

4. INTEGRAÇÃO COM IA

Após calcular indicadores:

Chamar:

POST /api/sindico/chat

Enviar:

{
tipo: "financeiro",
saldo: valor,
inadimplencia: percentual,
score: numero
}

IA deve retornar:

* análise financeira
* riscos
* recomendações

Salvar resposta em insights
