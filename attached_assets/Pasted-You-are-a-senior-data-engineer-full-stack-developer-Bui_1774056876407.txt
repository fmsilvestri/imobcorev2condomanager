📉 FORECAST ENGINE (IMPORTANT)

Create forecast.js:

Implement simple prediction:

- moving average
- trend detection

Example:

function forecast(values){
  const avg = values.reduce((a,b)=>a+b,0)/values.length;
  const trend = values[values.length-1] > avg ? "up" : "down";

  return {
    next: avg * 1.1,
    trend
  };
}
