## 4. INTEGRAÇÃO DI — Claude API

Em `NovaOSForm.tsx` etapa 4 e em `OSDetail.tsx`, chamar:

```typescript
async function getDiAnalysis(os: Partial<OrdemServico>, historico: OrdemServico[]) {
  const res = await fetch('https://api.anthropic.com/v1/messages', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      model: 'claude-sonnet-4-20250514',
      max_tokens: 600,
      system: `Você é Di, síndica virtual do condomínio ${condominioNome} em Florianópolis. 
               Analise a OS em até 3 parágrafos curtos. 
               Sugira prestador baseado no histórico, estime risco de recorrência e recomende prazo.
               Seja objetiva e use dados reais fornecidos.`,
      messages: [{
        role: 'user',
        content: `Nova OS: ${JSON.stringify(os)}\nHistórico de OSs desta categoria: ${JSON.stringify(historico.slice(-5))}`
      }]
    })
  })
  const data = await res.json()
  return data.content[0].text
}
```

Exibir com efeito typewriter no card `di-suggestion` (classes já existentes no CSS).
