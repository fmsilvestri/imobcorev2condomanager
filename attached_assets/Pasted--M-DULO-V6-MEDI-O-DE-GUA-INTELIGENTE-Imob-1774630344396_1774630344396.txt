# 🧱 ESTRUTURA DO BANCO (SUPABASE)

## 1. HIDRÔMETROS

CREATE TABLE IF NOT EXISTS hidrometros (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  condominio_id UUID,
  unidade_id UUID,
  tipo TEXT CHECK (tipo IN ('individual','geral')),
  numero_serie TEXT,
  localizacao TEXT,
  ativo BOOLEAN DEFAULT true,
  created_at TIMESTAMP DEFAULT NOW()
);

---

## 2. LEITURAS

CREATE TABLE IF NOT EXISTS leituras_agua (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  condominio_id UUID,
  hidrometro_id UUID REFERENCES hidrometros(id),
  leitura_anterior NUMERIC,
  leitura_atual NUMERIC,
  consumo NUMERIC,
  data_leitura DATE,
  origem TEXT DEFAULT 'manual', -- manual | iot
  created_at TIMESTAMP DEFAULT NOW()
);

---

## 3. RATEIO

CREATE TABLE IF NOT EXISTS rateio_agua (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  condominio_id UUID,
  competencia DATE,
  consumo_total NUMERIC,
  valor_total NUMERIC,
  tipo_rateio TEXT, -- igualitario | consumo | hibrido
  created_at TIMESTAMP DEFAULT NOW()
);

---

## 4. RATEIO POR UNIDADE

CREATE TABLE IF NOT EXISTS rateio_unidades (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  rateio_id UUID REFERENCES rateio_agua(id),
  unidade_id UUID,
  consumo NUMERIC,
  valor NUMERIC
);

---

## 5. ALERTAS DE CONSUMO

CREATE TABLE IF NOT EXISTS alertas_agua (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  condominio_id UUID,
  unidade_id UUID,
  tipo TEXT, -- vazamento | alto_consumo | anomalia
  descricao TEXT,
  nivel TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);

---

# ⚙️ LÓGICA DO SISTEMA

## 📥 Registrar leitura

- Busca última leitura
- Calcula consumo automaticamente:
  consumo = leitura_atual - leitura_anterior

---

## 💰 RATEIO AUTOMÁTICO

### 1. Consumo total:
- Soma das unidades OU hidrômetro geral

### 2. Diferença:
perda = consumo_geral - soma_individual

### 3. Regras:

IGUALITÁRIO:
- divide valor total igualmente

CONSUMO:
- proporcional ao consumo individual

HIBRIDO:
- 70% consumo
- 30% igualitário

---

## 🧠 DETECÇÃO DE VAZAMENTO

Se:
- consumo > média histórica * 1.5
→ gerar alerta "vazamento provável"

Se:
- consumo > 0 com imóvel vazio
→ alerta crítico

---

## 🔮 PREVISÃO DE CONSUMO

Base:
- últimos 3 meses

Calcular:
- tendência (subindo / descendo)
- previsão próximo mês
