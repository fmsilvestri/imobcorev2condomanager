Implementar o design completo do Módulo Água (AguaModule) do ImobCore v2
com dashboard de reservatórios, gráficos interativos e dados reais do Supabase/SSE.

=== STACK ===
React + TypeScript, Tailwind CSS, Chart.js (via npm install chart.js react-chartjs-2),
Supabase (tabelas existentes: reservoirs, sensor_readings, water_readings),
SSE já implementado no servidor Replit para tempo real.

=== HOOK useReservoirData.ts ===
Estado gerenciado:
  interface ReservoirState {
    reservoirs: Reservoir[]          // tabela reservoirs
    latestReadings: SensorReading[]  // último reading por reservoir_id
    history: HistoryPoint[][]        // array por sensor, pontos ordenados por created_at
    loading: boolean
    error: string | null
  }

Busca inicial (useEffect uma vez):
  1. GET /api/reservoirs → lista todos reservoirs do condominio_id ativo
  2. Para cada reservoir, GET /api/water-readings?reservoir_id=X&limit=200
     → guardar em history[i] ordenado por created_at ASC
  3. Calcular latestReadings[i] = history[i][history[i].length - 1]

SSE em tempo real:
  const es = new EventSource('/api/sse')
  es.onmessage = (e) => {
    const data = JSON.parse(e.data)
    if (data.type === 'water_reading') {
      // atualizar latestReadings e history do reservoir correspondente
      // recalcular agregações
    }
  }
  cleanup: es.close() no return do useEffect

Agregações calculadas no hook (useMemo):
  totalCapacity    = sum(r.capacity_liters)
  totalVolume      = sum(latestReadings[i].volume_litros ?? calcVolume(r, reading))
  avgLevel         = (totalVolume / totalCapacity) * 100
  consumoHoje      = history filtrado por today, diferença máx-mín volume
  alertCount       = count(latestReadings onde nivel < 40%)
  warnCount        = count(latestReadings onde nivel >= 40 e < 70%)

Cálculo de volume_litros quando nulo (lógica já existente no projeto):
  const heightCm = reading.nivel_cm  // ou distance invertida do sensor
  const pct = Math.min(heightCm / reservoir.height_cm, 1)
  const volume = Math.round(pct * reservoir.capacity_liters)
